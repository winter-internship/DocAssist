# Functional Specifications (기능 명세서)

본 문서는 `DocAssist` 프로젝트의 프론트엔드 소스 코드를 분석하여 작성된 기능 명세서입니다.
사용자(User) 및 관리자(Admin) 관점에서의 핵심 기능과 세부 요구사항을 정의합니다.

---

## 1. 인증 및 계정 관리 (Authentication & Account)

### 1.1 회원가입 (Sign Up)
- **위치**: `SignupPage.vue`
- **입력 정보**:
  - 이름 (Name): 필수 입력
  - 이메일 (Email): 이메일 형식 검증 (`@` 포함)
  - 비밀번호 (Password): 8자 이상, 확인 비밀번호와 일치 여부 검증
- **약관 동의**: 이용약관 및 개인정보 처리방침 동의 체크박스 (필수)
- **프로세스**: 입력 값 유효성 검사 -> API 요청 -> 로그인 페이지로 이동

### 1.2 로그인 & 로그아웃 (Login & Logout)
- **위치**: `LoginPage.vue`, `Sidebar` (공통)
- **기능**:
  - 이메일/비밀번호 인증 및 JWT 토큰 관리 (`localStorage`)
  - 관리자 이메일 목록에 포함된 경우 `ADMIN` 권한 부여
  - 로그아웃 시 토큰/세션 파기 및 로그인 화면 리다이렉트

### 1.3 비밀번호 찾기 및 재설정 (Password Recovery)
- **위치**: `ForgotPasswordPage.vue`, `ResetPasswordPage.vue`
- **비밀번호 찾기**: 가입 이메일 입력 시 재설정 링크 발송 (Mock API)
- **비밀번호 재설정**:
  - 이메일 링크를 통해 토큰(`token`)을 포함한 URL로 진입
  - 새 비밀번호 입력 및 확인 (유효성 검사)
  - 성공 시 로그인 페이지로 자동 이동

### 1.4 프로필 및 설정 (Profile & Settings)
- **위치**: `ProfilePage.vue`
- **정보 조회**: 프로필 이미지(이니셜), 가입일/최근접속일 표시
- **AI 이해 보조 설정**:
  - **난이도 조절**: 약/중/강 (단순화 강도)
  - **용어 설명 깊이**: Lv.1 ~ Lv.5
  - **근거 표시 방식**: 인라인/패널/호버 선택
- **UI 설정**: 테마(Light/Dark), 글자 크기, 문장 모드(구분선) 설정

## 2. 홈 및 문서 관리 (Home & Document Management)

### 2.1 홈 대시보드 (User Dashboard)
- **위치**: `DashboardHome.vue` (로그인 후 첫 화면)
- **기능**:
  - **환영 메시지**: 사용자 이름 표시 및 간편 업로드 영역(Dropzone) 제공
  - **최근 활동(Recent)**: 최근에 업로드/분석한 문서 카드 리스트 (클릭 시 이동)
  - **빠른 작업(Quick Actions)**: 문서 업로드, 드라이브 이동, Q&A 시작 바로가기 버튼

### 2.2 문서 업로드 (Upload)
- **위치**: `UploadPage.vue`
- **기능**: Drag & Drop 또는 파일 선택 (PDF, JPG, PNG). 업로드 완료 시 문서 분석 화면으로 자동 이동.

### 2.3 내 드라이브 (My Drive)
- **위치**: `DrivePage.vue`
- **목록/검색**: 제목/유형/상태별 필터링, 정렬(최신/오래된/이름순)
- **관리**: 다중 선택 후 일괄 삭제/내보내기, 개별 이름 변경

### 2.4 문서 비교 및 이해 (Doc Compare) ✨ 핵심 기능
- **위치**: `DocComparePage.vue`
- **화면 구성**: 좌측(원문) - 우측(이해 보조) 패널 분할 뷰
- **주요 기능**:
  - **동기 스크롤(Sync Scroll)**: 좌우 패널 스크롤 위치 동기화 (Toggle 가능)
  - **문단/규칙 보기**: 우측 패널을 '문단별 쉬운 설명' 또는 '핵심 규칙(Rule) 요약' 모드로 전환
  - **용어 하이라이트**: 본문 내 전문 용어 자동 하이라이팅
  - **용어 툴팁/상세**: 하이라이트 클릭 시 상세 설명(정의, 쉬운 말, 근거) Drawer 표시
  - **문서 Q&A**: 하단 채팅창을 통해 현재 문서 내용에 대해 즉시 질문 가능 (추천 질문 제공)

## 3. 지식 및 Q&A (Knowledge & Q&A)

### 3.1 용어집 (Glossary)
- **위치**: `TermsPage.vue`
- **기능**:
  - 문서에서 추출된 모든 전문 용어 목록 조회
  - **필터링**: 문서별, 태그별(법/보안/재무 등) 필터, 정렬(빈도/가나다/최신)
  - **상세 보기**: 용어 선택 시 우측 패널에 정의, 쉬운 설명, 근거 문장 표시
  - **핀(Pin)**: 중요 용어 상단 고정

### 3.2 AI 문서 Q&A
- **위치**: `QaPage.vue`
- **기능**:
  - 드라이브 내 문서를 선택하여 심층 대화 진행
  - **근거(Citation)**: 답변 내 근거 칩 클릭 시 원문 패널 하이라이트
  - **핀(Pin)**: 중요 답변 저장 및 모아보기

## 4. 관리자 시스템 (Admin System)

### 4.1 관리자 대시보드 (Admin Dashboard)
- **위치**: `AdminPage.vue` (Tab: Dashboard)
- **현황**: 사용자/문서 통계, 시스템 상태(Health Check), 작업 큐(Queue) 모니터링

### 4.2 운영 관리
- **사용자 관리**: 목록 조회, 권한(User/Admin) 변경, 계정 정지/활성
- **문서 관리**: 전체 문서 상태 조회, 실패 문서 재처리(Re-queue)
- **티켓 관리**: 문의/신고 사항 접수 및 상태 처리(접수/진행/완료)

### 4.3 시스템 설정
- **권한 정책**: 기능별 접근 제어(RBAC) 정책 관리
- **전역 설정**: AI 모델, 워커 동시성, 업로드 제한 설정 등

---

## 5. 공통 UI/UX 요소
- **Sidebar**: 페이지 네비게이션, 반응형(모바일 햄버거 메뉴)
- **Toast**: 작업 결과 알림 (우측 하단)
- **Responsive**: 데스크탑(Grid) / 태블릿 / 모바일(Stack) 최적화 레이아웃
