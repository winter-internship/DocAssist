## 📦 실행 방법

### 1단계: 의존성 설치

터미널에서 프로젝트 루트 디렉토리에서 실행:

```bash
npm install
```

**설명:**
- `package.json`에 정의된 모든 라이브러리를 다운로드합니다
- Vue, Vite, TypeScript 등이 설치됩니다
- 처음 한 번만 실행하면 됩니다

**예상 소요 시간:** 1-2분

---

### 2단계: 개발 서버 실행

```bash
npm run dev
```

**설명:**
- Vite 개발 서버가 시작됩니다
- 코드 변경 시 자동으로 새로고침됩니다 (Hot Module Replacement)
- 서버를 중지하려면 `Ctrl + C`를 누르세요

**예상 출력:**
```
  VITE v5.0.0  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

---

### 3단계: 브라우저에서 확인

브라우저를 열고 다음 주소로 접속:

```
http://localhost:3000
```

**예상 화면:**
- 상단: "문서 이해 보조 시스템" 헤더
- 왼쪽: Hero 섹션 (문서 설명)
- 오른쪽: 오늘의 요약 통계 카드
- 중간: 최근 업로드 문서 목록
- 중간: 빠른 작업 버튼들
- 하단: 최근 활동 목록

## 🔧 문제 해결

### 문제 1: `npm install` 오류

**증상:**
```
npm ERR! code ENOENT
npm ERR! syscall open
```

**해결:**
- Node.js가 설치되어 있는지 확인: `node --version`
- Node.js 18 이상 버전이 필요합니다
- [Node.js 다운로드](https://nodejs.org/)

---

### 문제 2: 포트 3000이 이미 사용 중

**증상:**
```
Error: listen EADDRINUSE: address already in use :::3000
```

**해결:**
1. 다른 프로그램이 3000번 포트를 사용 중입니다
2. `vite.config.ts`에서 포트를 변경하거나
3. 사용 중인 프로그램을 종료하세요

**포트 변경 방법:**
`vite.config.ts` 파일에서:
```typescript
server: {
  port: 3001,  // 3000 대신 다른 포트 사용
  // ...
}
```

---

### 문제 3: 화면이 하얗게 보임

**해결:**
1. 브라우저 개발자 도구 열기 (F12)
2. Console 탭에서 오류 확인
3. 오류 메시지를 확인하고 알려주세요
